# Passkey testing client

This is a simple client for testing passkey (webauthn) server.

> [!IMPORTANT]  
> Development is at a very early stage.

## Limitations:

- Only simple passkey creation process is supported.
- No attestation statement support.
- Only ES-256 algorithm is supported.

## Usage

Please refer to the `registration_test.go` file for an example of how to use the client.
